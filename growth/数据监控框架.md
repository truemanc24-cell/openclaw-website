### 1.2 指标定义表

| 级别 | 指标名称 | 定义 | 计算公式 | 目标值 (Day 20) | 目标值 (Day 60) |
|------|---------|------|---------|---------------|---------------|
| **一级** | DAU | 日活跃用户数 | 当日完成≥1 次 Agent 执行的用户 | 100 | 1000 |
| **一级** | MRR | 月收入 | 订阅收入 + 广告收入 | ¥500 | ¥5000 |
| **一级** | 用户增长率 | 净增长率 | (新用户 - 流失)/总用户 | 20%/周 | 15%/周 |
| **二级** | 日 UV | 独立访客数 | 当日访问网站的独立 IP | 1000 | 5000 |
| **二级** | 注册转化率 | 访问→注册 | 注册数/UV | 15% | 20% |
| **二级** | 激活率 | 完成新手引导 | 激活数/注册数 | 40% | 50% |
| **二级** | 7 日留存 | 7 天后回访 | D7 活跃/D1 注册 | 15% | 25% |
| **二级** | 付费转化率 | 活跃→付费 | 付费数/活跃数 | 8% | 10% |
| **二级** | K 因子 | 裂变系数 | 邀请新用户数/活跃用户 | 0.5 | 1.0 |

## 三、数据采集方案

### 3.1 埋点规范

| 事件类型 | 事件名 | 触发时机 | 必填参数 | 选填参数 |
|---------|-------|---------|---------|---------|
| **页面浏览** | page_view | 页面加载完成 | page_name, page_url | referrer, utm_source |
| **点击** | click | 元素点击 | element_id, element_name | page_name |
| **注册开始** | signup_start | 点击注册按钮 | method (email/github/wechat) | utm_source |
| **注册完成** | signup_complete | 注册成功 | user_id, method, duration | utm_source |
| **激活完成** | activation_complete | 完成新手引导 | user_id, steps_completed, duration | - |
| **付费** | purchase | 支付成功 | user_id, plan, amount, currency | coupon |
| **分享** | share_click | 点击分享 | user_id, channel | - |
| **邮件打开** | email_open | 邮件加载 | user_id, email_id, campaign_id | - |
| **邮件点击** | email_click | 邮件链接点击 | user_id, email_id, link_url | - |

### 3.2 技术实现

**前端埋点** (JavaScript):
```javascript
// 通用事件上报
function track(eventName, properties) {
  const payload = {
    event: eventName,
    properties: properties,
    timestamp: Date.now(),
    user_id: getCurrentUserId(),
    session_id: getSessionId()
  };
  
  // 发送到分析服务
  fetch('/api/track', {
    method: 'POST',
    body: JSON.stringify(payload)
  });
}

// 页面浏览
track('page_view', {
  page_name: 'homepage',
  page_url: window.location.href,
  referrer: document.referrer
});

// 点击事件
document.querySelectorAll('[data-track]').forEach(el => {
  el.addEventListener('click', () => {
    track('click', {
      element_id: el.dataset.trackId,
      element_name: el.dataset.trackName
    });
  });
});
```

**后端埋点** (Node.js):
