# 快速开始指南

## 5 分钟上手 OpenClaw

### Step 1: 安装

```bash
npm install -g openclaw
```

### Step 2: 初始化

```bash
openclaw init
```

### Step 3: 配置 Agent

编辑 `~/.openclaw/openclaw.json`，添加你的 Agent 配置。

### Step 4: 启动

```bash
openclaw gateway start
```

### Step 5: 连接飞书

在飞书开放平台创建 Bot，配置 App ID 和 Secret。
