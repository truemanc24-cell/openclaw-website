## 📊 使用 Excel 预测模型

打开 `revenue-forecast.csv` 文件，可以：

1. **查看预测数据**: Day 1-60 每日收入明细
2. **调整假设**: 修改"核心假设"工作表的参数
3. **自动计算**: 所有公式已嵌入，修改后自动更新
4. **敏感性分析**: 测试不同增长率/转化率的影响

### 关键公式

```
免费用户 (Day N) = 免费用户 (Day N-1) × (1 + 5%)
Pro 用户 = 免费用户 × 转化率 (2%-5%)
订阅收入 = Pro×29 + Enterprise×999 + Ultimate×4999
广告收入 = 免费用户 × 3 PV × 30 天 × ¥25 RPM ÷ 1000
总成本 = ¥708 + 收入×10%
净利润 = 总收入 - 总成本
```
