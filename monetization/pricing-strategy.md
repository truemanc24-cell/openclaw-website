## 5️⃣ 盈亏平衡分析

### 5.1 盈亏平衡点计算

**公式**:
```
盈亏平衡点 = 固定成本 ÷ (平均客单价 - 可变成本/单)
```

**假设**:
- 固定成本：¥708/月
- 平均客单价：¥500（混合订阅 + 服务）
- 可变成本/单：¥100

**计算**:
```
盈亏平衡点 = 708 ÷ (500 - 100) = 708 ÷ 400 = 1.77 ≈ 2 单/月
```
