## 一、免费图片生成工具

## 二、免费视频生成工具

### 1. RunwayML ⭐⭐⭐⭐

**免费额度**:
- 一次性 125 积分（不可再生）
- 125 积分 ≈ 25 秒 Gen-4 Turbo 视频
- 免费计划永久有效

**限制**:
- 积分用完需付费（无每月免费刷新）
- 视频有水印
- 最多 3 个项目
- 5GB 存储空间
- 无法使用 Gen-4 等高级模型

**注册链接**: https://runwayml.com/

**推荐指数**: ⭐⭐⭐（适合体验，长期使用成本高）

---

## 四、本地部署方案

### 1. Stable Diffusion WebUI (Automatic1111) ⭐⭐⭐⭐⭐

**成本**: 完全免费（开源）

**硬件要求**:
- GPU: NVIDIA 显卡（推荐 RTX 3060 12GB 或更高）
- 显存: 最低 4GB，推荐 8GB+
- 内存: 16GB+
- 硬盘: 50GB+（模型文件较大）

**优点**:
- 完全免费，无限制
- 社区支持好，插件丰富
- 可生成任意数量图片
- 可商用

**缺点**:
- 安装配置较复杂
- 需要一定技术基础
- 仅支持图片生成

**安装链接**: https://github.com/AUTOMATIC1111/stable-diffusion-webui

**推荐指数**: ⭐⭐⭐⭐⭐（长期使用的最佳选择）

---

### 2. ComfyUI ⭐⭐⭐⭐⭐

**成本**: 完全免费（开源）

**硬件要求**:
- 与 SD WebUI 类似
- 对显存优化更好（可在 4GB 显存运行）

**优点**:
- 节点式工作流，灵活度高
- 性能优化好，生成速度快
- 支持视频生成（SVD 等）
- 可自定义工作流

**缺点**:
- 学习曲线陡峭
- 界面不如 WebUI 友好
- 需要理解节点逻辑

**安装链接**: https://github.com/comfyanonymous/ComfyUI

**推荐指数**: ⭐⭐⭐⭐⭐（高级用户首选）

## 六、开发需求建议

### 现有工具不足

1. **免费视频生成额度太少**
   - RunwayML 仅一次性 125 积分
   - Pika Labs 每日限制严格
   - 建议：开发批量生成工具，组合多个免费平台

2. **中文配音质量参差不齐**
   - ElevenLabs 中文效果一般
   - 阿里云需付费
   - 建议：集成多个 TTS API，自动选择最优

3. **工作流分散**
   - 图片、视频、音乐需多平台切换
   - 建议：开发统一工作流管理工具

4. **免费额度管理复杂**
   - 各平台额度计算方式不同
   - 建议：开发额度追踪工具，提醒重置时间

### 建议开发的工具

1. **社交媒体内容生成器**
   - 输入主题 → 自动生成图片 + 视频 + 配音 + 背景音乐
   - 整合多个免费 API
   - 自动轮询使用各平台免费额度

2. **免费额度管理面板**
   - 追踪各平台剩余额度
   - 提醒每日/每月重置
   - 优化使用策略

3. **本地部署一键安装包**
   - 整合 SD WebUI + ComfyUI + 必要模型
   - 简化安装流程
   - 提供中文教程

## 七、总结

### 最佳免费组合（立即开始）

```
图片：Bing Image Creator + Leonardo.ai（每日约 100-200 张）
视频：剪映/CapCut（完全免费）+ Pika Labs（每日补充）
配音：剪映文本朗读（中文）+ ElevenLabs（英文）
音乐：YouTube Audio Library（免版权）
```

### 长期最佳方案（投资本地部署）

```
图片：Stable Diffusion WebUI（本地，无限制）
视频：剪映 + ComfyUI SVD（本地）
配音：ElevenLabs 付费套餐（¥35/月）
音乐：免版权音乐库组合
```

### 关键建议

1. **优先使用剪映**：功能全面，免费额度充足，适合社交媒体
2. **注册多个平台**：分散使用，最大化免费额度
3. **考虑本地部署**：长期使用成本最低，无限制
4. **注意版权**：免费≠可商用，注意各平台许可协议
