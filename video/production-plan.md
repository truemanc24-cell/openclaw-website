## 📋 执行摘要

本方案为 OpenClaw 网站制作宣传视频和教程视频提供完整的技术栈选型、工作流设计和成本预算。基于对阿里云百炼、RunwayML、OpenAI 等多模态模型 API 的调研，推荐采用**阿里云百炼为主 + 剪映专业版**的混合方案，实现高质量、低成本的视频制作。

## 1️⃣ 多模态模型能力调研

### 1.1 阿里云百炼（推荐）

#### 通义万相 - 图片生成
- **模型**: 通义万相 2.6
- **能力**: 文生图、图像编辑、复杂文本渲染
- **优势**: 中文理解优秀、价格亲民、国内访问速度快
- **适用场景**: 视频封面、教程插图、分镜草图

#### 通义千问 - 语音合成 (TTS)
- **能力**: 文本转语音、多音色选择、情感控制
- **优势**: 中文发音自然、支持方言、可定制音色
- **适用场景**: 视频配音、教程旁白

#### 通义万相 - 视频生成
- **模型**: 万相 2.6 视频生成大模型
- **能力**: 15 秒超长视频、电影级超写实人像
- **优势**: 专业影视创作级别、支持复杂场景
- **限制**: 单次生成时长有限，需后期拼接

#### 定价参考（估算）
| 服务 | 计价单位 | 价格（约） |
|------|---------|-----------|
| 文生图 | 每张 | ¥0.1-0.5 |
| TTS | 每 1000 字 | ¥5-15 |
| 视频生成 | 每秒 | ¥0.5-2.0 |

## 2️⃣ 视频制作工作流

### 2.1 完整工作流

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  脚本生成   │ →  │   配音      │ →  │   分镜      │ →  │   剪辑      │ →  │   发布      │
│  (Qwen)     │    │  (TTS)      │    │  (万相)     │    │  (剪映)     │    │  (多平台)   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### 2.2 各环节详细说明

#### 阶段 1: 脚本生成 (1-2 天)
**工具**: 通义千问 / Qwen3.5-Plus

**步骤**:
1. 确定视频主题和目标受众
2. 使用 Qwen 生成视频脚本大纲
3. 细化脚本内容，包括旁白文本、画面描述
4. 审核并优化脚本

**Prompt 示例**:
```
你是一位专业的视频脚本作家。请为 OpenClaw 智能助手平台
创作一个 3 分钟的宣传视频脚本，包括：
1. 开场（15 秒）- 吸引注意力
2. 问题引入（30 秒）- 痛点描述
3. 解决方案（60 秒）- 功能展示
4. 案例演示（45 秒）- 实际使用场景
5. 结尾（30 秒）- 行动号召

要求：语言生动、节奏明快、适合 B 站/YouTube 风格
```

#### 阶段 2: 配音制作 (0.5-1 天)
**工具**: 阿里云 TTS / 剪映文本朗读

**步骤**:
1. 将脚本旁白文本导入 TTS 工具
2. 选择合适的音色（推荐：温暖专业男声/女声）
3. 生成音频文件
4. 调整语速、停顿、情感
5. 导出 WAV/MP3 格式

**参数建议**:
- 语速：1.0-1.2x
- 音量：标准化到 -16 LUFS
- 格式：WAV 48kHz/24bit（后期用）

#### 阶段 3: 分镜生成 (1-2 天)
**工具**: 通义万相 / RunwayML

**步骤**:
1. 根据脚本拆分场景（每 5-10 秒一个镜头）
2. 为每个场景编写图像生成 Prompt
3. 批量生成场景图片
4. 筛选最佳结果
5. 必要时使用视频生成 API 创建动态镜头

**Prompt 示例**:
```
现代科技感办公室，明亮的自然光，程序员在使用 AI 助手，
电脑屏幕上显示代码和对话界面，专业摄影，电影级布光，
16:9 画幅，高清细节 --ar 16:9 --v 6
```

#### 阶段 4: 视频剪辑 (2-3 天)
**工具**: 剪映专业版（推荐）/ Adobe Premiere

**步骤**:
1. 导入所有素材（音频、图片、视频片段）
2. 根据配音节奏排列画面
3. 添加转场效果
4. 添加字幕（剪映可自动识别）
5. 添加背景音乐和音效
6. 调色和画面优化
7. 导出最终视频

**剪映 AI 功能**:
- ✅ 智能剪口播
- ✅ 文本朗读（备用 TTS）
- ✅ 智能字幕
- ✅ AI 降噪
- ✅ 超清画质增强
- ✅ 智能抠像

#### 阶段 5: 发布与分发 (0.5 天)
**平台**: 
- 国内：B 站、抖音、微信视频号、小红书
- 国际：YouTube、Twitter/X、LinkedIn

**优化建议**:
- 制作不同尺寸版本（16:9、9:16、1:1）
- 编写 SEO 友好的标题和描述
- 添加相关标签
- 制作封面缩略图

### 3.3 API 配置步骤（阿里云百炼）

#### 步骤 1: 开通服务
1. 访问 https://www.aliyun.com/product/bailian
2. 注册/登录阿里云账号
3. 开通大模型服务平台百炼
4. 完成实名认证

#### 步骤 2: 获取 API Key
1. 登录百炼控制台：https://bailian.console.aliyun.com/
2. 进入「API-KEY 管理」
3. 创建新的 API Key
4. 安全保存（不要提交到代码库）

#### 步骤 3: 配置环境
```bash
# 设置环境变量
export DASHSCOPE_API_KEY="your-api-key-here"

# 安装 Python SDK
pip install dashscope
```

#### 步骤 4: 调用示例

**文生图（通义万相）**:
```python
import dashscope
from dashscope import ImageSynthesis

dashscope.api_key = "your-api-key"

response = ImageSynthesis.call(
    model="wanx2.1",
    prompt="现代科技感办公室，程序员使用 AI 助手",
    size="1024*1024",
    n=1
)

# 保存生成的图片
import requests
img_url = response.output.results[0].url
img_data = requests.get(img_url).content
with open('output.png', 'wb') as f:
    f.write(img_data)
```

**文本转语音**:
```python
from dashscope import SpeechSynthesis

dashscope.api_key = "your-api-key"

response = SpeechSynthesis.call(
    model="sambert-zhichu-v1",
    text="欢迎使用 OpenClaw 智能助手",
    voice="zhichu",
    format="wav"
)

# 保存音频文件
with open('output.wav', 'wb') as f:
    f.write(response.get_audio_data())
```

**文本生成（脚本创作）**:
```python
from dashscope import Generation

dashscope.api_key = "your-api-key"

response = Generation.call(
    model="qwen-plus",
    prompt="请为 OpenClaw 创作一个 3 分钟宣传视频脚本...",
    max_tokens=2000
)

print(response.output.text)
```
