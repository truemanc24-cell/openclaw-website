## 🚀 第一阶段任务分配 (Day 1-3)

### 🌐 web-builder (小筑)

**任务 1: 网站框架搭建**
```
目标：使用 VitePress 搭建网站框架
截止时间：Day 3
交付物：可访问的网站 Demo

执行步骤:
1. 创建 VitePress 项目
2. 配置主题和样式
3. 部署到 Vercel
4. 配置自定义域名（可选）
```

**任务 2: 订阅系统配置**
```
目标：集成邮件订阅功能
截止时间：Day 7
交付物：可用的订阅表单

执行步骤:
1. 配置 SendGrid 或 ConvertKit
2. 创建订阅表单
3. 集成到网站
4. 测试邮件发送
```
