## 💡 核心特性

- 🌐 **多渠道支持**: WhatsApp、Telegram、Discord、iMessage 等
- 🤖 **多 Agent 协作**: 工作/生活/学习分离
- 🛠️ **丰富技能**: 文件操作、网络搜索、浏览器控制
- 🔒 **安全可靠**: 自托管、数据可控

## 🚀 立即开始

```bash
# 1. 安装
npm install -g openclaw

# 2. 运行向导
openclaw onboard

# 3. 打开控制面板
openclaw dashboard
```
