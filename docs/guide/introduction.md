# 介绍

OpenClaw 是一个强大的 AI 助手平台，运行在你的本地环境中。

## 核心特性

### 🤖 多 Agent 系统
- **Main Agent**: 主要助手，负责任务分发和协调
- **Deep Think**: 深度推理和分析
- **Dev**: 技能开发
- **Social**: 社交运营
- **Lingling**: 创意策划
- **Learning**: 学习助手
- **Sales**: 销售运营
- **Cece**: 增长策略

### 🛠️ 工具集成
- Feishu（飞书）完整集成
- GitHub CLI 集成
- 浏览器自动化
- 本地文件操作
- 语音合成（TTS）
- 图像处理

### 🧠 智能特性
- 自我改进工作流
- 记忆系统（短期 + 长期）
- 心跳机制（主动检查）
- 技能系统（可扩展）

## 架构概览

```
~/.openclaw/
├── agents/           # Agent 配置
│   └── main/agent/   # Main Agent 工作区
├── skills/           # 技能库
├── extensions/       # 扩展插件
└── workspace/        # 工作空间
```
