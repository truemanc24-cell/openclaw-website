# 关于 OpenClaw

## 项目愿景

OpenClaw 致力于打造一个强大、安全、可扩展的本地 AI 助手平台。

## 设计理念

### 🎯 实用主义
- 优先解决实际问题
- 清晰的步骤胜过冗长的解释
- 稳定胜过速度

### 🔒 安全第一
- 本地优先，保护隐私
- 不擅自修改系统配置
- 修改前先备份

### 🧠 持续学习
- 从错误中学习
- 自我改进工作流
- 社区共享技能

## 核心功能

| 功能 | 描述 |
|------|------|
| 多 Agent 协作 | 8 个专业 Agent 各司其职 |
| 技能系统 | 可扩展的技能库 |
| Feishu 集成 | 完整的飞书 API 支持 |
| GitHub 集成 | 通过 gh CLI 管理项目 |
| 浏览器自动化 | 控制浏览器执行任务 |
| 记忆系统 | 短期 + 长期记忆 |
| 心跳机制 | 主动检查和提醒 |

## 技术栈

- **运行时**: Node.js 18+
- **框架**: VitePress（文档）
- **模型**: 支持多种 LLM（通义千问、Claude 等）
- **扩展**: Feishu、GitHub、浏览器等
