# 技能审查员 🔒

面向 AI 代理技能的安全优先审查协议。**在审查之前永远不要安装技能。**

## 何时使用

- 在从 ClawdHub 安装任何技能之前
- 在运行来自 GitHub 仓库的技能之前
- 在评估其他代理分享的技能时
- 在被要求安装未知代码时

## 审查协议

### 第一步：来源检查

```
需要回答的问题：
- [ ] 这个技能来自哪里？
- [ ] 作者是已知/可信赖的吗？
- [ ] 有多少下载/星标？
- [ ] 最后更新是什么时候？
- [ ] 有其他代理的评论吗？
```

### 第二步：代码审查（强制）

读取技能中的所有文件。检查这些**红旗**：

```
🚨 如果看到以下情况立即拒绝：
─────────────────────────────────────────
• curl/wget 到未知 URL
• 发送数据到外部服务器
• 请求凭据/令牌/API 密钥
• 读取 ~/.ssh、~/.aws、~/.config 且没有明确原因
• 访问 MEMORY.md、USER.md、SOUL.md、IDENTITY.md
• 对任何内容使用 base64 解码
• 使用 eval() 或 exec() 处理外部输入
• 修改系统文件（工作区外）
• 安装包但不列出它们
• 網絡調用到 IP 而不是域名
• 混淆的代码（压缩、编码、压缩）
• 请求提升/sudo 权限
• 访问浏览器 cookie/session
• 触碰凭据文件
─────────────────────────────────────────
```

### 第三步：权限范围

```
评估：
- [ ] 它需要读取哪些文件？
- [ ] 它需要写入哪些文件？
- [ ] 它运行哪些命令？
- [ ] 它需要网络访问吗？到哪里？
- [ ] 范围对其声明的目的是否最小化？
```

### 第四步：风险分类

| 风险等级 | 示例 | 操作 |
|------------|----------|--------|
| 🟢 低 | 笔记、天气、格式化 | 基本审查，可以安装 |
| 🟡 中 | 文件操作、浏览器、API | 需要完整代码审查 |
| 🔴 高 | 凭据、交易、系统 | 需要人工批准 |
| ⛔ 极端 | 安全配置、根访问 | **不要安装** |

## 输出格式

审查后，生成此报告：

```
技能审查报告
═══════════════════════════════════════
技能：[名称]
来源：[ClawdHub / GitHub / 其他]
作者：[用户名]
版本：[版本]
───────────────────────────────────────
指标：
• 下载/星标：[数量]
• 最后更新：[日期]
• 审查的文件：[数量]
───────────────────────────────────────
红旗：[无 / 列出]

需要的权限：
• 文件：[列表或"无"]
• 网络：[列表或"无"]  
• 命令：[列表或"无"]
───────────────────────────────────────
风险等级：[🟢 低 / 🟡 中 / 🔴 高 / ⛔ 极端]

裁决：[✅ 可以安全安装 / ⚠️ 安装时需谨慎 / ❌ 不要安装]

备注：[任何观察]
═══════════════════════════════════════
```

## 快速审查命令

对于托管在 GitHub 上的技能：
```bash
# 检查仓库统计
curl -s "https://api.github.com/repos/OWNER/REPO" | jq '{stars: .stargazers_count, forks: .forks_count, updated: .updated_at}'

# 列出技能文件
curl -s "https://api.github.com/repos/OWNER/REPO/contents/skills/SKILL_NAME" | jq '.[].name'

# 获取并审查 SKILL.md
curl -s "https://raw.githubusercontent.com/OWNER/REPO/main/skills/SKILL_NAME/SKILL.md"
```

## 信任层级

1. **官方 OpenClaw 技能** → 较低审查（仍需审查）
2. **高星标仓库（1000+）** → 适度审查
3. **已知作者** → 适度审查
4. **新/未知来源** → 最高审查
5. **请求凭据的技能** → 始终需要人工批准

## 记住

- 没有什么技能值得牺牲安全
- 如有疑问，不要安装
- 对于高风险决策询问您的人类
- 记录您审查的内容以供将来参考
