# 天气技能

**技能名称**: `weather`  
**版本**: 1.0.0  
**来源**: [ClawHub](https://clawhub.ai/)

---

## 📖 技能介绍

天气技能提供实时天气查询和预报功能，无需 API key，开箱即用。

### 核心功能

- 🌤️ **实时天气** - 当前温度、湿度、风速
- 📅 **天气预报** - 3 天/7 天预报
- 🌍 **多城市** - 支持全球城市查询
- ⚠️ **天气预警** - 恶劣天气提醒
- 📊 **详细数据** - 紫外线、能见度、降水概率

---

## 🚀 快速开始

### 1. 安装技能

```bash
clawhub install weather
```

### 2. 无需配置

天气技能使用 wttr.in 服务，**无需 API key**，安装即可使用。

## 🔧 高级用法

### 自定义输出格式

```bash
# 简洁模式
curl wttr.in/Beijing?format=3

# 详细模式
curl wttr.in/Beijing?T
```

### 定时天气提醒

```javascript
// 每天早上 8 点发送天气
cron.add({
  schedule: { kind: "cron", expr: "0 8 * * *" },
  payload: { kind: "systemEvent", text: "查询北京天气" }
});
```

### 天气预警

```javascript
// 检测恶劣天气
if (weather.condition.includes("暴雨") || weather.condition.includes("台风")) {
  message.send({ text: "⚠️ 恶劣天气预警！" });
}
```

## ⚠️ 注意事项

### 数据源

- **服务**: wttr.in
- **更新频率**: 每小时更新
- **覆盖范围**: 全球城市

### 速率限制

- 免费使用，无严格限制
- 建议单个 IP 不超过 60 次/分钟

### 常见问题

**Q: 城市找不到？**  
A: 使用标准城市名，如 "Beijing" 或 "北京"

**Q: 数据不准确？**  
A: wttr.in 使用多个数据源，可能有延迟

**Q: 支持英文城市名吗？**  
A: 支持，推荐使用拼音或英文
