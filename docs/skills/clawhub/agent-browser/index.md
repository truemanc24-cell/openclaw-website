## 📖 技能介绍

无头浏览器自动化 CLI，为 AI agent 优化，支持可访问性树快照和基于 ref 的元素选择。

### 核心功能

- 🌐 **浏览器自动化** - 控制无头浏览器
- 🌳 **可访问性树** - 获取页面结构
- 🎯 **元素选择** - 基于 ref 精准选择
- 📸 **页面快照** - 捕获页面状态
