## 📖 技能介绍

使用 mcporter CLI 直接列出、配置、认证和调用 MCP 服务器/工具（HTTP 或 stdio）。

### 核心功能

- 🔌 **MCP 服务器** - 连接 Model Context Protocol 服务器
- ⚙️ **配置管理** - 编辑 MCP 配置
- 🔐 **认证处理** - 管理 API 认证
- 🛠️ **CLI 工具** - 命令行直接调用
