## 📖 技能介绍

使用 Nano Banana Pro (Gemini 3 Pro Image) 生成/编辑图像。

### 核心功能

- 🎨 **文生图** - 根据文字描述生成图像
- 🖼️ **图生图** - 基于现有图像编辑
- 📐 **多分辨率** - 支持 1K/2K/4K
- ✏️ **图像编辑** - 修改现有图像
