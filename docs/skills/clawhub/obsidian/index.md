## 📖 技能介绍

通过 obsidian-cli 自动化 Obsidian 笔记库（纯 Markdown 笔记）。

### 核心功能

- 📝 **笔记创建** - 自动创建 Markdown 笔记
- 🔍 **内容搜索** - 在笔记库中搜索
- 🔗 **链接管理** - 管理笔记间的双向链接
- 📊 **图谱查看** - 查看知识图谱

---
