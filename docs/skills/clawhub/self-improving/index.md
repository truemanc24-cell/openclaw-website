## 📖 技能介绍

自我反思 + 自我批评 + 自我学习 + 自组织记忆的完整自我改进系统。

### 核心功能

- 🤔 **自我反思** - 评估自己的工作
- ⚠️ **自我批评** - 发现并纠正错误
- 📚 **自我学习** - 持续学习新知识
- 🗂️ **自组织记忆** - 结构化存储经验
