## 5.1 访问控制面板

### 打开方式

**方法 1：CLI 命令（推荐）**

```bash
openclaw dashboard
```

**效果**：自动在默认浏览器中打开控制面板。

**输出示例**：
```
Opening dashboard...
✓ Dashboard available at: http://127.0.0.1:18789
✓ Opening in default browser...
```

**方法 2：手动访问**

```bash
# 确保网关正在运行
openclaw gateway status

# 在浏览器中访问
http://127.0.0.1:18789
```

**方法 3：指定端口**

```bash
# 如果网关使用非标准端口
openclaw dashboard --port 18790

# 或在浏览器中访问
http://127.0.0.1:18790
```

### 浏览器访问

**支持的浏览器**：

| 浏览器 | 版本要求 | 推荐度 |
|--------|----------|--------|
| Chrome | 90+ | ⭐⭐⭐⭐⭐ |
| Firefox | 88+ | ⭐⭐⭐⭐⭐ |
| Safari | 14+ | ⭐⭐⭐⭐ |
| Edge | 90+ | ⭐⭐⭐⭐ |
| Opera | 76+ | ⭐⭐⭐ |

**推荐**：使用最新版本的 Chrome 或 Firefox 获得最佳体验。

### 远程访问配置

**默认情况**：控制面板仅本地访问（`127.0.0.1`）

**允许远程访问**：

```json
{
  "gateway": {
    "port": 18789,
    "bind": "0.0.0.0",  // 监听所有网络接口
    "auth": {
      "mode": "token",
      "token": "your-secure-token"
    }
  }
}
```

**重启网关**：
```bash
openclaw gateway restart
```

**从其他设备访问**：
```
http://<你的 IP 地址>:18789
例如：http://192.168.1.100:18789
```

**使用 Tailscale 安全访问**：

```bash
# 配置 Tailscale（参考教程 03）
openclaw onboard --tailscale

# 访问地址
https://your-hostname.tailnet.ts.net:18789
```

### 安全注意事项

| ⚠️ 安全提示 | 说明 |
|------------|------|
| **不要开放到公网** | 除非有强认证和 HTTPS |
| **使用强 Token** | 至少 16 位随机字符 |
| **启用防火墙** | 限制访问 IP 范围 |
| **使用 HTTPS** | 远程访问时必须 |
| **定期更换 Token** | 建议每 90 天更换 |

**配置防火墙（Linux）**：
```bash
# 仅允许本地访问
sudo ufw deny 18789

# 允许特定 IP
sudo ufw allow from 192.160.1.0/24 to any port 18789

# 查看规则
sudo ufw status
```

**配置防火墙（macOS）**：
```bash
# 系统偏好设置 → 安全性与隐私 → 防火墙
# 添加 OpenClaw 或限制端口访问
```

### 热重载说明

**什么是热重载？**

热重载（Hot Reload）指不重启网关的情况下应用新配置。

**支持热重载的配置**：
- ✅ 渠道配置（添加/删除渠道）
- ✅ Agent 配置（切换模型）
- ✅ 会话设置
- ✅ 工具策略

**不支持热重载的配置**：
- ❌ 网关端口
- ❌ 网关绑定地址
- ❌ 认证模式
- ❌ 工作空间路径

**热重载操作**：

```
[保存并应用] 按钮

✓ 配置已保存
✓ 配置已热重载
✓ Telegram 渠道已更新
✓ 新模型已生效
```

**需要重启的情况**：

```
⚠️  以下修改需要重启网关：

- 网关端口：18789 → 18790
- 绑定地址：127.0.0.1 → 0.0.0.0

[立即重启] [稍后手动重启]
```

## 代码示例汇总

### 1. 打开控制面板
```bash
openclaw dashboard
```

### 2. 指定端口
```bash
openclaw dashboard --port 18790
```

### 3. 远程访问配置
```json
{
  "gateway": {
    "port": 18789,
    "bind": "0.0.0.0",
    "auth": {
      "mode": "token",
      "token": "your-secure-token"
    }
  }
}
```

### 4. 防火墙配置（Linux）
```bash
sudo ufw allow from 192.168.1.0/24 to any port 18789
sudo ufw status
```

### 5. 重启网关
```bash
openclaw gateway restart
```

### 6. 查看状态
```bash
openclaw gateway status
openclaw channels status
openclaw nodes status
```

### 7. 查看日志
```bash
openclaw logs --follow
openclaw logs --limit 100
openclaw logs --grep "telegram"
```

### 8. 配置验证
```bash
openclaw doctor
openclaw configure --validate
```

### 9. 导入导出配置
```bash
# 导出
openclaw config export --output backup.json

# 导入
openclaw config import --file backup.json
```

### 10. 技能管理
```bash
# 列出技能
openclaw skills list

# 安装技能
openclaw skills install stock-tracker

# 启用/禁用
openclaw skills enable web-search
openclaw skills disable image-analysis

# 卸载技能
openclaw skills uninstall weather
```
