# Vercel 推出 AI 网站托管方案，开发者可一键部署 AI 应用

**发布日期**: 2026-03-27  
**来源**: Vercel 官方博客  
**阅读时间**: 4 分钟

---

## 📰 新闻摘要

Vercel 今日推出全新的 AI 网站托管方案，支持开发者一键部署 AI 应用。新方案包括：
- AI 模型集成（支持 OpenAI、Anthropic、阿里云等）
- 自动扩缩容（应对流量高峰）
- 边缘计算（降低延迟）
- 免费额度（每月 100 万次推理）

这一举措将大幅降低 AI 应用的部署门槛。

---

## 🎯 核心功能

### 1. AI 模型集成

**支持的模型提供商**:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- 阿里云百炼 (Qwen)
- Google (Gemini)
- 自定义模型

**配置示例**:
```json
{
  "ai": {
    "provider": "bailian",
    "model": "qwen3.5-plus",
    "apiKey": "${BAILIAN_API_KEY}"
  }
}
```

### 2. 自动扩缩容

**特点**:
- 根据流量自动调整资源
- 支持突发流量（10 倍峰值）
- 按实际使用付费

**优势**:
- 无需手动配置服务器
- 降低运维成本
- 保证用户体验

### 3. 边缘计算

**部署位置**:
- 全球 100+ 边缘节点
- 自动选择最近节点
- 延迟 < 50ms

**适用场景**:
- 实时对话应用
- 全球用户访问
- 低延迟要求

## 🚀 快速开始

### 部署 OpenClaw 网站

```bash
# 1. 克隆项目
git clone https://github.com/yourname/opclaw-website

# 2. 进入目录
cd opclaw-website

# 3. 部署到 Vercel
vercel deploy

# 4. 配置环境变量
vercel env add BAILIAN_API_KEY
```

### 配置 AI 功能

```json
// vercel.json
{
  "ai": {
    "provider": "bailian",
    "model": "qwen3.5-plus"
  },
  "regions": ["hnd1", "sin1", "pdx1"]
}
```
