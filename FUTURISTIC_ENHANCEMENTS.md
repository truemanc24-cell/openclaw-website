## 🎨 核心特效

### 1. 增强粒子系统 (ParticleBackground.vue)

#### 主要特性
- **100+ 彩色粒子** - 青色、绿色、紫色、粉色、蓝色多色混合
- **智能连线** - 粒子间自动连接，形成科技感网络
- **三角形连接** - 粒子网络中自动填充半透明三角形
- **阴影发光** - 每个粒子带有蓝色光晕
- **旋转动画** - 粒子自身旋转效果
- **大小渐变** - 粒子大小动态变化 (1-6px)
- **透明度动画** - 粒子呼吸效果

#### 交互效果
- **鼠标追踪** - 鼠标移动时生成发光粒子尾迹
- **气泡效果** - 鼠标悬停时粒子放大
- **抓取连线** - 靠近鼠标时粒子连线增强

#### 技术实现
```javascript
// 粒子配置亮点
particles: {
  number: { value: 100 },
  size: { min: 2, max: 6 },
  links: {
    distance: 180,
    opacity: 0.4,
    triangles: { enable: true, opacity: 0.15 }
  },
  shadow: {
    enable: true,
    blur: 10,
    color: '#00d9ff'
  }
}
```

### 2. 动态光源系统 (Light Sources)

#### 三层光源
1. **青色主光源** (400px) - 位于左侧，最强光源
2. **紫色辅助光源** (350px) - 位于右侧
3. **绿色强调光源** (300px) - 位于中心

#### 光源动画
- **自主移动** - 光源在页面内缓慢漂浮
- **边界反弹** - 触达边界自动改变方向
- **脉冲效果** - 光源大小和透明度周期性变化
- **色相旋转** - 整体色相轻微变化增加动感

#### CSS 变量同步
```javascript
// 实时更新 CSS 变量
document.documentElement.style.setProperty(
  `--light-${i}-x`, 
  `${source.x * 100}%`
)
```

## ⚡ 性能优化

### 优化措施
1. **分层渲染** - 粒子、光源、网格分层
2. **Canvas 渲染** - 高性能粒子绘制
3. **requestAnimationFrame** - 流畅动画
4. **CSS 硬件加速** - transform 和 opacity
5. **混合模式** - `mix-blend-mode: screen` 高效合成

### 性能建议
- 粒子数量：50-150（根据设备性能）
- FPS 限制：60fps
- 动画时长：避免过快的循环

## 🚀 使用方式

### 在页面中使用

```vue
<script setup>
import TechLayout from './components/TechLayout.vue'
import ParticleBackground from './components/ParticleBackground.vue'
</script>

<template>
  <TechLayout>
    <template #particles>
      <ParticleBackground />
    </template>
    
    <div class="content">
      <!-- 你的内容 -->
    </div>
  </TechLayout>
</template>
```

### 自定义配置

#### 调整粒子数量
编辑 `ParticleBackground.vue`:
```javascript
number: {
  density: { area: 700 }, // 减小 = 更多粒子
  value: 100
}
```

#### 调整光源速度
编辑 `ParticleBackground.vue`:
```javascript
const lightSources = ref([
  { x: 0.2, y: 0.3, vx: 0.0003, vy: 0.0002 } // 增大速度值
])
```

#### 修改配色
编辑 `vars.css`:
```css
:root {
  --vp-c-brand: #你的颜色;
  --vp-c-accent: #你的颜色;
}
```

## 🔧 故障排除

### 粒子不显示
- 检查 tsparticles 是否安装：`npm list tsparticles-slim`
- 查看浏览器控制台是否有错误
- 确认组件正确注册

### 性能问题
- 减少粒子数量
- 降低动画复杂度
- 禁用部分特效（移动端）

### 光源不移动
- 检查 CSS 变量是否正确设置
- 确认 JavaScript 无报错
- 查看 animateLightSources 是否运行
