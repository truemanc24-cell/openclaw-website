## 🔧 替换广告单元 ID

### 当前代码位置
`docs/.vitepress/theme/components/AdUnit.vue`

### 需要替换的位置

```vue
<!-- 顶部横幅 -->
data-ad-slot="1234567890"  ← 替换为实际 ID

<!-- 侧边栏 -->
data-ad-slot="2345678901"  ← 替换为实际 ID

<!-- 内容插页 -->
data-ad-slot="3456789012"  ← 替换为实际 ID

<!-- 底部 -->
data-ad-slot="4567890123"  ← 替换为实际 ID
```

### 替换步骤
1. 复制 AdSense 后台的广告单元 ID
2. 打开 `AdUnit.vue` 文件
3. 替换 4 个 `data-ad-slot` 的值
4. 保存并提交 Git
5. 部署到 Vercel

## 💡 临时方案（审核期间）

在 AdSense 审核期间，可以：

### 选项 A：显示占位符（当前）
- 粉色渐变背景
- 文字提示"广告位（审核通过后显示广告）"

- 缺点：无收益

### 选项 B：移除广告位
- 暂时隐藏广告组件
- 审核通过后再添加
- 优点：页面更干净
- 缺点：需要两次部署

### 选项 C：使用其他广告
- 先接入其他广告联盟
- AdSense 通过后再切换
- 优点：提前开始收益
- 缺点：可能违反 AdSense 政策

**推荐**: 选项 A（当前方案）

## 📊 广告收益追踪

### AdSense 后台指标
- **展示次数**: 广告被展示的次数
- **点击次数**: 用户点击广告的次数
- **CTR**: 点击率（点击/展示）
- **CPC**: 每次点击费用
- **收入**: 累计收益

### 预期数据（技术类网站）
| 指标 | 预期值 |
|------|--------|
| CTR | 1-3% |
| CPC | $0.10-0.50 |
| RPM | $5-15（每千次展示收入） |
